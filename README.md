# 打印成本&售价计算器
自用的打印成本、售价核算工具，支持纸张、黑白/彩色单独计价，可根据打印覆盖率计算墨水消耗，生成打印计价单。

✅ 在线体验：https://print.id5.fun
⚡ 运行环境：纯前端

## 项目说明
本项目是为宿舍打印店日常运营开发的自用小工具，已开源供有需要的同学参考使用。
- 代码风格：非专业开发，以实用为主，代码实现较为简洁直接
- 精度说明：覆盖率计算通过将PDF转换为150PPI图片分析得出，结果仅供参考，不保证专业级精准度
- 适用场景：个人/小型打印店成本核算，介意精度的专业场景请勿使用

## 核心功能
- 📄 纸张成本、黑白/彩色打印价格单独设置
- 🖨️ 按打印覆盖率智能估算墨水/碳粉消耗成本
- 💰 自动计算单页/整单总成本、利润及建议售价
- 📑 生成可复制/导出的打印计价单
- 🌐 纯前端运行，无需部署后端服务，开箱即用

## 技术栈
- 前端框架：Vue 3
- 样式方案：Tailwind CSS
- PDF处理：PDF.js
- 运行环境：纯HTML/CSS/JavaScript
